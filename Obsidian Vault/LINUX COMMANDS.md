# cd, ls 

- cd:
     cd ..  :parent directory(goes one directory up)
     cd - : goes to the parent directory
 - ls:
     ls -a: to show all files
     ls -l: timestamps etc and other detailed info of files
     ls -R to show the details of files with in directories 
     ls -r: to show the details of files in reverse order of arrangement


# file,cat,less,history

- file:
     used to the type of file and some other details 

- less: 
      read the file to search for details etc 
- ctrl+R:
       to manually search for previous commands until a match is found

- history:
        to see the history of commands

# copy commands ,mv,renaming,removing

- cp:
     to cp a file to particular directory
     eg:
     cp ameen.python one
- cp -r:
     to copy a directory and its files recursively
- mv:
     to move  a file and also renaming a file
     also to move directories and its by default recursive
     mv -i: trigger before overwriting existing file
     mv -b: two create a back up while moving an existing in the required directory
- rm:
        to remove files and directories;
        rm -f: to remove files by force; no trigger
        rm -i: triggered removal
        rm -r/ rmdir: to recursivley delete a direc
